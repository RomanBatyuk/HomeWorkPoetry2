# HomeWorkPoetry

## Описание:

HomeWorkPoetry - это виджет банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:
```
git@github.com:RomanBatyuk/HomeWorkPoetry2.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование:

1. Откройте проект в PyCharm.
2. Теперь вы можете поучавствовать в разработке ;)
